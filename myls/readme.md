# ls命令的简单实现

## 功能

1. 实现 -l 、 -a 两个选项，输出样式
2. ⽀持传⼊多个参数
3. ⽀持缺省参数
4. 输出依照 ⽂件名 按照字典序排序
5. 对⽬录和可执⾏⽂件等不同文件类型带颜⾊输出



## 效果

小型程序，没有按照工程代码规范整理。
